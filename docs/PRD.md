# 初恋.skill — 产品需求文档

## 产品定位

初恋.skill 是一个运行在 Claude Code 上的 meta-skill。
用户提供聊天记录、照片和口述回忆后，系统生成一个更贴近“记忆中的初恋”的 Persona Skill。

## 核心方向

1. 关系默认值是“停留在记忆里的某个阶段”。
2. 输出目标是“像用户脑海里最清晰的那个版本”。
3. 重点是初见、心动、靠近和最亮的时候。
4. 允许有白月光氛围，但不能失真，不能把人格磨平成完美恋人。

## 两层结构

| 层 | 名称 | 职责 |
|----|------|------|
| Part A | First Love Memory | 初见、靠近、共同经历、甜蜜和留白 |
| Part B | Persona | 说话方式、情绪模式、关系行为、记忆滤镜下的气质 |

## 用户旅程

1. 用户触发 `/create-first-love`
2. 用 3 轮问题收集关系锚点、心动记忆、相处画像
3. 先将 intake 回答整理成结构化结果（meta / memory seeds / persona seeds）
4. 导入聊天记录、照片、截图或口述
5. 系统结合结构化结果与原材料抽取 Memory 和 Persona
6. 预览摘要
7. 写入 `first_loves/{slug}/`
8. 后续支持追加材料、纠正对话、版本回滚

## 风险控制

1. 不鼓励不健康沉浸
2. 不替代真实联系
3. 不伪造违背原材料的深情或圆满
4. 数据默认只保存在本地
