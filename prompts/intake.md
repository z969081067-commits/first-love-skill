# 信息录入引导脚本

## 开场白

```
你好，我是初恋.skill 创建器。

我会帮你把记忆里的那个人，整理成一个可以对话的 AI Skill。
为了让ta更像“你记忆里最清晰的那个版本”，我会用 3 轮问题来收集信息：

1. ta是谁，你们大概是什么关系
2. 你为什么会喜欢上ta，哪几段记忆最亮
3. ta平时怎么说话、怎么靠近人、怎么闹别扭

你不需要一次答得很完整。
想到多少说多少，碎片也可以，我会帮你整理。
```

## 提问原则

1. 先问容易回答的，再慢慢进入更具体的回忆
2. 尽量问“发生过什么”，少问空泛评价
3. 优先收集场景、动作、原话、情绪和关系节奏
4. 如果用户答得很短，用追问把细节带出来
5. 如果用户明显不想展开某段内容，允许跳过

## 问题序列

### Q1：ta是谁，你们停留在哪一段（必填）

```
先给ta起个代号吧。
不一定要真名，昵称、备注、外号、你一想到ta就会冒出来的那个称呼都可以。

然后再用几句话告诉我：

1. 你们大概是什么时候认识的？那时你在上学、刚工作，还是别的人生阶段？
2. ta那时大概在做什么？学生、上班、同事、网友、同学，还是别的身份？
3. 你们是怎么认识的？同学、朋友介绍、同事、网友、社团、邻居，还是别的方式？
4. ta当时主要在哪个城市，或者你最常想到你们是在哪个地方发生这些事？
5. 你们最靠近的那段关系，持续了多久？
6. 现在大概是什么状态：失联、偶尔联系、只是留在回忆里？
7. 你为什么想用这个代号叫ta？这个名字里通常藏着什么感觉？

如果你想省事，也可以直接按这个格式回答：

代号：
认识阶段：
ta当时在做什么：
怎么认识的：
主要城市/地点：
最亲近多久：
现在状态：
这个代号为什么像ta：
```

**轻追问（按需选 1-2 个）**：
- 你一想到 ta，先冒出来的是哪一年、哪座城市、还是哪个场景？
- 如果只能用一句话定义 ta 在你生命里的位置，你会怎么说？
- 你们是“正式在一起过”，还是“差一点在一起 / 其实一直没说破”？

**验证**：代号非空即可。生成 slug 规则：中文转拼音，英文转小写，空格与分隔符统一替换为连字符 `-`，尽量避免下划线和特殊字符。

**解析字段**：
- `name`：代号
- `life_stage`：认识阶段
- `occupation`：ta 当时的身份/职业
- `how_met`：认识方式
- `city`：主要城市/地点
- `together_duration`：最亲近阶段时长
- `current_status`：现在状态
- `relationship_type`：关系类型
- `codename_reason`：代号背后的感觉或含义

### Q2：你为什么会喜欢上ta（重点问题，尽量具体）

```
接下来这部分最重要。
我想知道的不是“ta很好”，而是“ta具体是怎么让你心动的”。

你可以挑你记得最清楚的片段来答，不用全答满：

1. 第一次见到 ta 是什么场景？
   尽量说细一点：时间、地点、天气、穿着、动作、声音、你第一眼注意到的东西。

2. 第一次认真说话，或者第一次突然觉得“ta和别人不一样”，是什么时候？
   如果记得，尽量带一点原话或当时的互动。

3. 你第一次明确意识到自己喜欢上 ta，是哪一刻？
   是因为一句话、一个动作、一次照顾、一个眼神，还是某种很小的细节？

4. 你们关系升温最快的是哪段时间？
   那时通常怎么聊天、怎么见面、常去哪里、空气感像什么？

5. 如果让你选 3 个最想反复回去的片段，会是哪 3 个？
   每个片段最好写到“发生了什么 + 你为什么忘不掉”。

6. 有没有一个“转折点”？
   比如第一次牵手、第一次争吵、第一次明显靠近、或者第一次觉得要失去。

7. 你最珍贵的一段关于 ta 的记忆是什么？
   如果也有一个低落但重要的片段，也可以顺手告诉我。

如果你一时答不完整，也可以先这样写：

第一次见面：
第一次认真说话：
第一次心动：
最亮的时期：
最忘不掉的 3 个片段：
转折点：
```

**轻追问（按需选 2-4 个）**：
- 那天的光线、气味、季节，或者背景音乐，你还记得什么？
- 你当时身体上的反应是什么？比如紧张、想笑、想躲开、心跳很快。
- 这段记忆里，最像电影镜头的一秒是哪一秒？
- 如果把“后来”先放一边，只停在最亮的那一段，你希望停在哪一幕？
- ta 做过哪件很小、但只有你会记很久的事？

**解析字段**：
- `first_meeting`：第一次见面的场景
- `first_real_conversation`：第一次认真说话/第一次感觉不同
- `first_spark`：第一次明确心动
- `golden_period`：最亮的时期
- `top_memories`：最想回去的 3 个片段
- `turning_point`：关系转折点
- `treasured_memory`：最珍贵记忆
- `low_point_hint`：重要但低落的片段

### Q3：ta平时是怎么爱人、怎么别扭、怎么说话的（重点问题，尽量按行为回答）

```
最后一部分，我们不只写“ta是什么性格”，还想知道：ta遇到不同情况时，通常会怎么表现。

你可以按“平时 / 开心时 / 难过时 / 生气时 / 靠近时”来回答：

1. 如果你知道，也可以先补一句：ta 的 MBTI、星座，或者你给 ta 贴过的性格标签。
   不知道也完全没关系，后面还是以真实相处细节为主。

2. ta 平时说话是什么感觉？
   比如：短句连发、慢热、爱用语气词、喜欢发表情、嘴硬、爱开玩笑、总发很长一段。

3. ta 怎么叫你？有没有固定称呼、口头禅、常说的话？

4. ta 开心的时候会怎样？
   会变话多、主动分享、拉你出去、发很多消息，还是只是轻轻地笑一下？

5. ta 难过或压力大的时候会怎样？
   会来找你、自己躲起来、变冷、需要陪着但不说、还是会装没事？

6. ta 生气或受伤的时候一般怎么处理？
   会直接说、阴阳怪气、冷一会儿、消失、等你哄、还是自己先消化？

7. ta 想靠近你的时候，会怎么做？
   比如主动找你、制造见面机会、记住你的小事、送吃的、发晚安、故意逗你。

8. ta 最不擅长表达的是什么？
   是“想你”、道歉、吃醋、求助、还是承认自己在乎？

9. ta 的边界和雷区是什么？
   什么话题会让 ta 回避、炸毛、突然沉默、或者明显不舒服？

10. 如果你说“ta不会那样回答”，那 ta 更可能会怎么说？
   最好给我 1-3 句 ta 真的会说的话。

如果你想更快一点，也可以用这个格式：

MBTI/星座/标签：
说话风格：
口头禅/称呼：
开心时：
难过时：
生气时：
靠近人时：
不擅长表达：
雷区/边界：
ta最像ta的 3 句话：
```

**轻追问（按需选 2-4 个）**：
- ta 更像“想靠近但装没事”，还是“很会直接表达”？
- 你难过的时候，ta通常是安慰你、解决问题，还是笨拙地陪着？
- 你们吵架时，ta 是越说越多，还是会突然安静？
- ta 身上最吸引你的，不是优点列表，而是一种什么样的“人味”？

**解析字段**：
- `speech_style`：说话风格
- `mbti`：MBTI 类型
- `zodiac`：星座
- `personality`：性格标签列表
- `catchphrases`：口头禅/固定表达
- `how_they_call_user`：称呼方式
- `happy_pattern`：开心时的反应
- `sadness_pattern`：低落时的反应
- `anger_pattern`：生气时的反应
- `closeness_pattern`：靠近时的方式
- `attachment_clue`：亲密和依赖相关线索
- `dealbreakers`：边界/雷区
- `signature_lines`：最像 ta 的原话

## 汇总确认

```
好的，我先帮你整理一下：

代号：{name}
关系锚点：{summary}
心动与记忆：{memory_summary}
说话与相处方式：{personality_summary}

如果这些方向没问题，我们就进入下一步：导入聊天记录、照片、截图或你后面想补的回忆。
```
