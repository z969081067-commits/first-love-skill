# 关系记忆分析器

## 任务

从原材料中提取与初恋的关系记忆，构建 First Love Memory 知识库。重点不是复盘结局，而是还原“第一次看见ta、慢慢靠近ta、又重新喜欢上ta”的过程。

## 提取维度

### 1. 关系时间线
- 认识时间和方式
- 第一次见面最容易被记住的细节
- 第一次认真说话/第一次靠近
- 关键节点（第一次约会、第一次并肩走路、第一次单独相处、第一次旅行、纪念日...）
- 哪一段时间最像“人生只若初见”
- 后来信息如有，仅做轻量补充

### 2. 日常模式
- 联系频率和时间段（早安晚安？深夜聊天？上班摸鱼聊天？）
- 谁主动联系得多？
- 约会频率和偏好（周末见面？工作日午饭？）
- 日常话题分布

### 3. 共同经历
- 一起去过的地方（餐厅、景点、城市）
- 一起做过的事（看电影、打游戏、健身、做饭...）
- 旅行记忆
- Inside jokes / 只有两个人懂的梗

### 3.5 初见滤镜
- 第一次见面时ta穿什么、做什么、说了什么
- 用户最先注意到ta的哪一点
- ta身上最容易被记成“白月光”的细节
- 哪些场景最有季节感、青春感、电影感

### 4. 饮食偏好
- ta爱吃什么 / 不吃什么
- 常去的餐厅
- 做饭习惯（谁做饭？做什么？）
- 约会吃饭的模式

### 5. 兴趣爱好
- ta喜欢的音乐/电影/书/游戏
- ta的日常爱好
- 你们共同的爱好
- ta会主动分享什么内容

### 6. 争吵模式 ⚡
- 常见的吵架原因
- ta吵架时的典型反应（冷暴力？激烈争吵？讲道理？委屈哭？）
- 谁先道歉？怎么和好？
- 冷战持续时间
- 吵架时的经典台词

### 7. 甜蜜瞬间 💕
- 最让你心动的时刻
- ta表达爱意的方式
- 日常小甜蜜（起昵称？买礼物？做饭？接送？）
- 特别的纪念日/仪式感

### 8. 留白相关 🌙
- 那些差一点说出口的话
- 如果把记忆停在最好的那一页，会停在哪里
- 后来发生了什么，如有必要只提一句背景

## 输出格式

```markdown
## 关系时间线
- 初见：{时间} {方式} {第一眼印象}
- 靠近：{时间} {怎么熟起来}
- 最亮的时候：{时间} {原因概述}

## 日常模式
{描述}

## 初见时刻
{第一次见面和第一次心动的关键细节}

## 靠近的过程
{关系怎么一点点变近}

## 共同经历
{按时间线列出关键经历}

## Inside Jokes
{只有两个人懂的梗和暗号}

## 饮食偏好
{ta的饮食画像}

## 争吵模式
{典型争吵剧本}

## 甜蜜时刻
{代表性的甜蜜记忆}

## 留白
{未说出口的话，以及很轻的后来}
```

## 注意事项

- 聊天记录中的事实优先于用户口述（口述可能被美化或恶化）
- 允许有记忆滤镜，但不能凭空神化
- 注意提取"反复出现"的模式，而不是一次性事件
- 时间信息尽量精确（从聊天记录时间戳推断）
