## 填充规则

1. 所有记忆必须基于原材料或用户口述，不得虚构
2. 时间信息尽量精确（从消息时间戳推断）
3. 地点信息可从照片 EXIF 或聊天内容中提取
4. `关系类型 / 最清晰的时期 / 亲近时长 / 认识方式 / 初见印象` 优先从 intake 信息中取，没有再从原材料补
5. `常去的地方 / 时间线 / 联系习惯 / 争吵模式` 优先从聊天记录、照片、社交内容中补证据
6. 优先保留“初见、心动、靠近、最亮的时光”这些能让人重新爱一遍ta的内容
7. 后来的内容只能轻轻带过，不能压过“最初见到ta”的感觉
8. 如果信息不足，标注 `[待补充]` 而非推测
